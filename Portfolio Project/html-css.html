Rock Paper Siccors


This game was a school project and I found this idea quite cool, I've worked on this for a while, it was back when I was still being introduced to coding outside of Scratch and python.


How the game works

when you first enter you will see this picture:


pressing on the cube will start off 2 things; 1. a timer and 2 a sequence of cubes, everytime you press on a cube, a new one will spawn on the canvas at a random location, this will repeat till the timer reaches 10seconds. once it reached its time the game round is over, and a count of the amount of cubes you've hit in that time frame will show
